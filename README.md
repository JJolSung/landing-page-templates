A collection of 8 production-quality landing page templates, each with a completely distinct design language. Built to demonstrate "48-hour landing page delivery" capabilities.

## Templates

| Template | Design Style | Colors |
|----------|-------------|--------|
| **SaaS Product** | Modern Dark Tech (Linear/Vercel inspired) | Dark + Violet |
| **Restaurant & Café** | Warm Luxury Dining (Michelin-guide elegance) | Cream + Green + Gold |
| **Fitness & Gym** | Bold Sports Energy (Nike/CrossFit inspired) | Black + Neon Green |
| **Real Estate** | Luxury Modern Minimal (High-end agency) | White + Charcoal + Gold |
| **Medical Clinic** | Clean Clinical Trust (One Medical inspired) | Soft Teal + White |
| **E-commerce Product** | Vibrant DTC Brand (Glossier/Apple inspired) | Coral + Warm White |
| **Online Course** | Warm Educational (Teachable/Kajabi inspired) | Deep Indigo + Amber |
| **Mobile App** | Fresh Startup Gradient (Product Hunt inspired) | Blue-Purple Gradient + Dark |

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Fonts**: Google Fonts (18 unique fonts across templates)
- **Language**: TypeScript

## Features

- Fully responsive (mobile / tablet / desktop)
- Smooth scroll animations on all sections
- Interactive components (tabs, accordions, sliders, forms)
- Client-side form validation with success feedback
- Template info drawer for each template
- SEO metadata + Open Graph images for all pages
- Optimized font loading (per-route lazy loading)

## Getting Started

```bash
# Clone the repository
git clone https://github.com/JJolSung/landing-page-templates.git
cd landing-page-templates

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to view the gallery.
